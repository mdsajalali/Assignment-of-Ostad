1. <!--...--> = Used to create comments in the HTML source code.
2. <a> =Used to create a hyperlink, allowing users to navigate to another web page or a specific location within the current page.
3. <address> = Used to define the contact information for the author or owner of a document or article.
4. <article> = Used to define an independent, self-contained piece of content, such as blog post or a news article. 
5. <audio> = Used to embed audio content, such as music or podcasts into an HTML document.
6. <b> = Used to specify bold text within an HTML document.
7. <blockquote>=Used to indicate a section of quoted content, typically rendered with indentation or some other visual distinction.
8. <body> = Used to define the main content of an HTML document, which is visible to the user.
9. < br /> = Used to insert a line break or a forced line break within a paragraph.
10 <button> = Used to create a clickable button within an HTML form or as part of a user interface.
11. <del> = Used to indicate that a specific text or content has been deleted or removed from a document.
12.<div> = Used to create a container or a division within a HTML document, which can be styled manipulated using CSS or JavaScript.
13. <header> = Used to define the introductory or navigational section at the top of a document or a section.
14. <h1> to <h6> = Used to define heading or titles of different levels, with <h1> being the highest level and <h6> being the lowest level.
15. <html> = Used to define the root element of an HTML document, encompassing the entire page.
16. <img> = Used to embed an image into an HTML document, with the “src” attribute specifying the image source.
17. <link> = Used to establish a link between an HTML document and an external resource, such as a stylesheet or an icon.
18. <menu> = Used to define a list of commands or options presented to the user, often used in the context of a contextual menu.
19. <p> = Used to define a paragraph of text within an HTML document.
20. <select> = Used to create a dropdown list or a selection menu, allowing users to choose one or more options from a list.
